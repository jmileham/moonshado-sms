= moonshado-sms

Moonshado SMS Gem

== Usage

Put this in Development and Test environment so no real SMS are sent out:
  Moonshado::Sms.config = {
    :test_env => true,
    :sms_api_url => nil
  }

Production environment config:
  Moonshado::Sms.config = {
    :sms_api_url => ENV['MOONSHADOSMS_URL']
  }
